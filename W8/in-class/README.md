## Activity 2: UI Layout in Miro (Optional)

Ideation is the process of putting design ideas into a visual form before building the actual website. UX/UI designers often use tools such as Miro to create wireframes and layouts before the design is handed over to web developers. This helps plan the structure, layout, and user experience of a webpage in a clear and organized way.

For this activity, you will use Miro to improve one of the webpages you created this week.

**Instruction:**
1. Choose one webpage that your team created during this week.
2. Open Miro and create a wireframe for a new and improved version of that webpage
3. Focus on improving:
layout
navigation
spacing
readability
overall user experience (UX)
4. Keep your design simple and clean.
5. Upload your Miro design as PDF into this repository
6. Use the wireframe to update your actual webpage in HTML and CSS.
7. Make necessary commits and push the updated webpage to your GitHub repository.
